# 🚗 Car Insurance Claim Prediction
 
A machine learning classification project to predict whether a car insurance policyholder will file a claim.
 
---
 
## 📋 Project Overview
 
| Question | Answer |
|---|---|
| **Target** | `OUTCOME` — 1 = filed a claim, 0 = did not file a claim |
| **One row represents** | A single car insurance policyholder |
| **Number of features** | ~18 features (demographic + driving behavior + vehicle info) |
| **Number of rows** | ~10,000 rows |
| **Model used** | Random Forest Classifier |
